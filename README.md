# 1. Introduction

**Read Gurbani Desktop Application** is a desktop application to read gurbani scripture.

# 2. Development

## Requirements

- git
- npm (latest version)

## Installation Steps

### Download code into your machine

```bash
git clone repository
```

### Install dependencies

```bash
npm install

npm run rebuild

```

### Run project 

```bash
npm start
```
